1. В HTML
 - Створити div з класом wrapper, 
 а в ньому ще один div з класом pulse

2. В CSS:
Тут у нас буде три правила і однин @keyframes 
для ефекту пульсування.

Для батьківського div:
 .wrapper {
      ширина: 100%;
      Висота: 100vh;
      фоновий колір: #D04746;
      ввімкнути флексбокси
      вертикальне вирівнювання з флексбоксів: center;
      горизонтальне вирівнювання з флексбоксівt: center;
    }
Для вкладеного div:
 .pulse {
      ввімкнути флексбокси
      вертикальне вирівнювання з флексбоксів: center;
      горизонтальне вирівнювання з флексбоксівt: center;
      ширина: 150px;
      висота: 150px;
      колір букв: gray;
      фоновий колір: white;
      радіус заокруглення: 50%;
      виклик анімації: radial-pulse 1s infinite;
      підкреслення: underline;
    }
Ефект наведення на саму кнопку:
 .pulse:hover{
      анімація: none;
      товщина букв: 900;
      підкреслення букв: none;
      трансформація: scale(1.1);
      затримка: all 1s;
      розмір букв: 1.2rem;
      колір букв: black;
      тінь для блоку: 0 0 40px rgb(0, 0, 0);

    }
Анімація:

тут-щось-трба-написати radial-pulse {
      0% {
        тінь для блоку: 0 0 0 0px rgba(0, 0, 0, 0.5);
      }

      100% {
        тінь бля блоку: 0 0 0 40px rgba(0, 0, 0, 0);
      }
    }
