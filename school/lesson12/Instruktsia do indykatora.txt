1. HTML 
Створити один зовнішній блок і в ньому в середині ще один
2. CSS
Для зовнішнього блока:
- ширина 200пікселів
- висота 30пікселів
- рамка: 3пікселі щирина суцільна #fff
- радіус заокруглення кутів - 7пікселів
- приховати переповнення
- зовнішній відступ - 40пікселів
- тінь: 0 3пікселі 4пікселі rgba(0,0,0,0.3)

Для внутрішнього блока:
- висота 30пікселів
- ширина 400пікселів
- фон - лінійний градієнт з повторенням
            background: repeating-linear-gradient(45deg, rgba(255,255,255,0.9) 0, rgba(255,255,255,0.9) 5px, transparent 5px, transparent 10px);
анфмація: назва 4секунди дінійна функція безпервне виконання


анімація {
	0%{
		зовнішній лівий відступ 0
	}
	50%{
		прозорість 0
	}
	100%{
		зовнішній лівий відступ -100%
	}

}
